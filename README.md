# Точка Роста

**Точка роста** — лендинг о внедрении инноваций и получении статуса резидента Сколково.

## Структура

- index.html — основной файл сайта
- /images — изображения
- favicon.ico — иконка
- robots.txt — правила для поисковиков
- sitemap.xml — карта сайта для поисковиков
